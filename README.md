## Features 

Manage your info displays from anywhere.

  * Observe
    * Display status
    * Uptime
    * CPU load
    * CPU temperature
    * RAM usage
  * Administer
    * Reboot Pi
    * Shutdown Pi
    * Restart browser
    * Display on / off
  * Configure your Raspberry Pi settings
    * Hostname
    * Display protocol
    * Display orientation
  * Plan with the advanced time schedule
    * Show a webpage
    * Reboot Pi
    * Shutdown Pi
    * Restart browser
    * Display on / off
    * Execute universal command
