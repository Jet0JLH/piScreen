## Features 

Manage your info displays from anywhere.

  * Show a webpage on your info display
  * Reload page
  * Restart browser / pi
  * Turn on info display or set it to standby
  * Use schedule for weekly commands
  * Get info about current state of your system
    * Display status
    * Pi load
